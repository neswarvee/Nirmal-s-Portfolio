IBM AI Group Project: Paradroid Game

Back to Projects

Project Overview

Collaborated effectively with a dynamic team to develop a tool for IBM that taught their users about AI.

Project Details

  • Developed a retro-style action game inspired by the original Paradroid, incorporating custom artwork and music.
  • Created a comprehensive requirements document using the MoSCoW prioritization technique, outlining must-have, should-have, could-have, and won't-have features agreed upon with the client.
  • Designed and implemented game mechanics including:
    • Quiz System: Players answer questions related to IBM Skills Build topics (AI, Data Science, Threat Hunting) to take control of enemies.
    • Shooting Mechanic: Introduced flexible shooting angles for enhanced gameplay.
    • Enemy Behavior: Implemented patrol and chase mechanics with dynamic voice lines.
    • Teleportation: Developed a teleport system between game floors with user-friendly animations and prompts.
  • Integrated accessibility features to ensure inclusivity for players with color blindness, along with a clear user manual.
  • Created a save/load system allowing players to maintain their progress.
  • Conducted extensive testing with beta testers, receiving positive feedback on gameplay and performance.
