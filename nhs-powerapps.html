Project Overview

Developed an intuitive PowerApps application for consultants at the Clatterbridge Cancer Trust, allowing them to manage their apps more effectively. The application provides a streamlined and user-friendly interface to enhance their workflow.

Key Features

  • Apps gallery with icons, names, and descriptions.
  • Favourite, request access, and run action buttons.
  • Search bar for quick app access.
  • Personalized request access functionality with confirmation pop-ups.

Technologies Used

  • Microsoft PowerApps
  • Power Automate
